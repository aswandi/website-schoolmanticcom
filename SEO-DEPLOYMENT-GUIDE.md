## 🚀 Langkah Deployment ke Hosting

### 1. Build Production
```bash
npm run build
```
Ini akan membuat folder `dist` dengan file production-ready.

### 2. Upload ke Hosting
Upload semua file dari folder `dist` ke public_html atau root directory hosting Anda:
- index.html
- assets/ (folder dengan CSS, JS, images)
- logo-schoolmantic.jpg
- foto/ (folder galeri)
- robots.txt
- sitemap.xml
- .htaccess

### 3. Konfigurasi Domain
Pastikan domain schoolmantic.com sudah pointing ke hosting Anda.

### 4. SSL Certificate
Aktifkan SSL/HTTPS di hosting (biasanya gratis via Let's Encrypt).
