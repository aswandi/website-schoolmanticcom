## 🚀 Cara Upload ke cPanel

### Langkah 1: Login cPanel
1. Buka: `https://yourdomain.com/cpanel` atau `https://yourdomain.com:2083`
2. Login dengan username & password hosting

### Langkah 2: File Manager
1. Klik **"File Manager"** di cPanel
2. Masuk ke folder **"public_html"**

### Langkah 3: Upload & Extract
1. Upload file `schoolmantic-ready-to-upload.zip`
2. Klik kanan pada file ZIP → **"Extract"**
3. Pindahkan semua file dari folder hasil extract ke root `public_html`
4. Hapus file ZIP dan folder kosong

### Langkah 4: Verify
1. Buka browser: `https://schoolmantic.com`
2. Test semua link dan tombol
3. Test halaman foto: `https://schoolmantic.com/foto`
